# 大陆 ECS 域名 ICP 备案指南（ailen.store）

> **实名认证 ≠ ICP 备案**  
> 阿里云账号实名只证明你是账号持有人；域名在**中国大陆服务器**对外提供网站，还需向工信部备案。

---

## 一、备案类型

| 类型 | 说明 |
|------|------|
| **首次备案** | 域名从未在工信部备过案 |
| **接入备案** | 域名已在别家备过案，现改用阿里云大陆 ECS 托管 |
| **新增网站** | 同一主体下已有备案号，新增 `ailen.store` |

个人博客一般选：**个人 + 网站**。

---

## 二、前置条件

1. 域名 `ailen.store` 已在阿里云完成 **实名认证**（域名控制台 → 实名认证成功）
2. 有一台 **中国大陆地域** ECS（如华东、华北），且能 SSH
3. 备案主体：**个人**（身份证）或 **企业**（营业执照）
4. 阿里云账号已完成 **实名认证**
5. 准备材料：
   - 身份证正反面
   - 阿里云 **备案专用幕布** 照片（APP 人脸识别或邮寄幕布，按控制台指引）
   - 个人：手机号、邮箱、通信地址

---

## 三、操作入口

1. 登录 [阿里云备案系统](https://beian.aliyun.com/)
2. 或：控制台 → **ICP 备案** → **开始备案**
3. 按向导填写：
   - **主体信息**：姓名、身份证号、地址
   - **网站信息**：
     - 域名：`ailen.store`
     - 网站名称：如「个人学习博客」（避免含「新闻」「论坛」等敏感词）
     - 网站内容：个人博客 / 学习笔记
   - **服务器**：选择你的 **大陆 ECS**（会自动关联 IP）

---

## 四、接入备案（域名已在别处备过）

若 `ailen.store` 之前在其他云商备过案：

1. 备案系统选 **接入备案**
2. 填写原 **备案号**（在工信部可查）
3. 提交后阿里云审核 → 管局审核
4. 通过后域名解析到大陆 ECS 即可访问，不再弹「备案不符合要求」

---

## 五、审核流程与时间

```
填写资料 → 阿里云初审（1～2 工作日）
        → 工信部短信核验（24 小时内回复）
        → 管局审核（约 7～20 工作日，各省不同）
        → 下发 ICP 备案号
```

备案期间：

- **域名可能仍无法访问**（或继续显示拦截页）
- **ECS 可用 IP 直接访问**调试（`http://你的IP`）
- 不要频繁改解析，按备案专员要求操作

---

## 六、备案通过后

1. 在网站 **页脚** 添加备案号链接，例如：
   ```html
   <a href="https://beian.miit.gov.cn/" target="_blank" rel="noopener">京ICP备xxxxxxxx号</a>
   ```
2. 解析 `ailen.store` → 大陆 ECS IP
3. 配置 HTTPS：`certbot --nginx -d ailen.store`
4. 公安备案（部分省份要求，控制台会提示）

---

## 七、常见问题

| 问题 | 说明 |
|------|------|
| 已实名仍被拦截 | 缺 **ICP 备案** 或 **接入备案** |
| 个人能备 `.store` 吗 | 可以，按个人网站提交 |
| 香港 ECS 要备案吗 | **不需要**（服务器不在大陆） |
| 备案能撤吗 | 可以注销；换香港机则无需大陆备案 |

---

## 八、与当前项目的关系

- 备案 **只解决域名访问拦截**，不修复前端 Vue 报错
- 备案期间继续用 **IP + build 最新前端** 调试
- 通过后使用 `https://ailen.store` 并更新 `application-prod.yml` 的 CORS
